Persons Table - Sirius Case Management

The persons table includes data on people within the Sirius system. It contains both LPA, Guardianship and Supervision data, as well as data relating to persons who are informed parties.

Stability: The persons table is to be considered unstable .

Analytical Platform : Yes

Alternative formats : JSON
