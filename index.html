OPG Data Dictionary

This is an early stage prototype for a data-dictionary for the Office of the Public Guardian.

This site contains information on data schemas, constrained lists, formats and their history. The aim is a resource that can help you discover what data we have, where it lives and what it means.

This is a very early prototype. Feedback is welcomed to the #opg-analytical-hookup channel on Slack.
